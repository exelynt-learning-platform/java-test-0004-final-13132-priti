Concentric Number Pattern in Java

This program prints a square pattern of numbers.

First, we take a variable n which decides the starting number of the pattern.

The size of the pattern is calculated as (2*n - 1).

Two nested loops are used to print rows and columns.

For each position, the minimum distance from the top, bottom, left, and right sides is calculated.

Then the number (n - min) is printed.

This creates a concentric square pattern where numbers decrease toward the center.
